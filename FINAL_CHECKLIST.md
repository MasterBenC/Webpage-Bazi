# 宸垚命馆上线前统一清单

当前本地版本已经具备：品牌官网、客户排盘、真太阳时、大运流年、DeepSeek 对话、客户档案、对话复核、提示词版本、知识库、SQLite 数据库与管理端权限。

下面事项需要在准备正式上线时统一处理，不影响当前继续试用。

## 需要项目方最终确认

- 正式品牌名称、Logo、公司或个体工商户主体。
- 咨询服务说明、价格、次数、退款规则和人工服务范围。
- 宸垚老师正式介绍、可公开经历和最终头像授权。
- 隐私政策、用户协议、免责声明的法律审核版本。
- 用于知识库的真实文章、案例和口播资料，并完成客户信息脱敏。

## 需要项目方申请或购买

- 正式域名。
- 云服务器或云应用平台。
- 中国大陆服务器所需的 ICP 备案。
- HTTPS 证书（多数云平台可免费配置）。
- 已认证微信公众号或微信开放平台账号（需要微信登录时）。
- 短信服务账号（需要手机号验证码时）。
- 微信支付或其他支付商户号（开始收费时）。

## 上线时需要配置

- 生产环境 DeepSeek API Key。
- 生产数据库和自动备份。
- 正式管理员密码与定期更换规则。
- API 限流、错误监控、日志保留和敏感数据脱敏。
- 域名、服务器、防火墙、HTTPS 与备份恢复演练。

## 推荐上线顺序

1. 内部试用与真实案例校验。
2. 小范围邀请客户免费测试。
3. 补齐协议、隐私政策和服务边界。
4. 配置正式域名、服务器与备份。
5. 再接登录、支付和商业套餐。
